# 如何将moka添加至其他群？我可以加moka为好友吗？

目前，你可以自由地将moka添加到任何群组，也能自由地添加moka为好友，moka会自动同意所有的加群邀请和好友申请。

向moka直接发送群二维码图片是错误的邀请方法，正确将moka拉入群的方法是使用群资料页面的群聊成员->邀请按钮（需要群主事先打开允许群成员邀请好友入群权限）。
