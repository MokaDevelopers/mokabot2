# 随机选择

用于在你不知所措的时候随机选择

```
moka选择 <选项1> [选项2] [选项3]
```

## 复习

<ClientOnly>
  <Messenger :messages="[
    { position: 'right', msg: 'moka选择 吃饭 睡觉 打游戏' },
    { position: 'left', msg: 'moka的选择是：吃饭' },
    { position: 'right', msg: 'moka选择 吃饭 睡觉 打游戏' },
    { position: 'left', msg: 'moka的选择是：睡觉' },
  ]"></Messenger>
</ClientOnly>
